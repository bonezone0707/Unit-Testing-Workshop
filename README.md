# Unit-Testing-Workshop
Unit Testing Workshop

Unit Testing Workshop
Programming Language:
Python
Testing Framework:
- I have used pytest, which is a popular unit testing framework, for testing some python scripts written for having automations. In order to develop the automation scripts I had to understand how to use pytest for testing several functionalities.

 How to Run
1. Install pytest
Pip install pytest

2. Open a terminal in the project folder.
3. Run: 
Pytest -v

Expected Result
Collected data from 5 tests

Pass 5

There were no failures.
